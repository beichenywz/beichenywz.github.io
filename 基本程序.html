5、循环语句
（1）语句结构：
for语句结构
for (初始化; 条件; 增值) { 执行; } //括号内可为空，“;”不可舍去
while语句结构
while (条件) { 执行; }
do-while语句结构
do { 执行; }while (条件); //先执行一次后再判断，等价于无条件执行一次的wihle语句
（2）关键词：
1
continue：跳过此轮循环
2
break：结束循环
关键词应用
for (int a = 1; a < 10; a++) { if (a == 5) { continue; } if (a == 8) { break; } cout << a << " "; } //输出为：1 2 3 4 6 7

数组定义

7、字符串
（1）定义
字符串定义
char a[10]; string s; //字符串类型，不限长度
（2）操作
字符串操作
cout << s[0]; //输出字符串s的第0个字符（从0开始） s += str; //在字符串s后拼接字符串str s < str //比较字符串s的字典序是否在字符串str的字典序之前 s.size(); 或 s.length(); //得到字符串s的长度 s.substr(pos, len) //截取字符串s，从第pos个字符开始的len个字符，并返回这个字符串 s.insert(pos, str) //在字符串s的第pos个字符之前，插入字符串str,并返回这个字符串 s.find(str,pos) /*在字符串s中从第pos个字符开始寻找str(字符或字符串)，并返回位置，找不到返回-1【需要强制转换为int类型才能直接输出-1，如：cout << (int)s.find(str);】，pos可省略。
